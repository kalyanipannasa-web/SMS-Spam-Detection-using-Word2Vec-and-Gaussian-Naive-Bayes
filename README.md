# SMS Spam Detection using Word2Vec and Gaussian Naive Bayes

This project implements SMS Spam Detection using NLP and Machine Learning techniques. Text messages are preprocessed using regex, stopword removal, and lemmatization. Word embeddings are generated using Word2Vec and converted to document vectors using Average Word2Vec. A Gaussian Naive Bayes classifier is trained to classify messages as Spam or Ham.

## Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Gensim
- Scikit-learn

## Workflow
1. Data Loading
2. Text Cleaning
3. Stopword Removal
4. Lemmatization
5. Word2Vec Training
6. Average Word2Vec
7. Gaussian Naive Bayes
8. Prediction & Evaluation

## Dataset
SMS Spam Collection Dataset

## Results
Achieved high accuracy in detecting spam messages using Average Word2Vec embeddings.
